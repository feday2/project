News Portal with article page and category page
-----------

This is demo projects on Symfony 4 framework.

Installation
------------

1. Clone repository

    ```bash
    $ git clone https://github.com/feday2/Symfony.git && cd Symfony
    ```
    
2. Install dependencies

    ```bash
    $ composer install
    ```
    
3. Run server

    ```bash
    $ php bin/console server:start
    ```
